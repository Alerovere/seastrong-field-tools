# Echosounder Bathymetry

## Overview

This tool processes depth data collected with a **Deeper Pro** echosounder and applies tide corrections using a reference tide gauge dataset. It is used in SEASTRONG to map the bathymetry of cross-shelf corridors encompassing mangrove forests, seagrass beds, and coral reefs, contributing to **Task 1.2** (mapping the physical environment within corridors) and **Task 4.5** (the role of spatial connectivity in coastal protection).

## Why we do this in SEASTRONG

Bathymetry data collected with the Deeper Pro echosounder feeds directly into two SEASTRONG work packages. In **WP1**, bathymetry maps are used to characterise the physical environment of cross-shelf corridors and to compute spatial connectivity metrics among mangrove forests, seagrass beds, and coral reefs. In **WP4**, the same data is used to set up and calibrate the hydrodynamic model (XBeach), which simulates wave attenuation and coastal protection under different levels of spatial connectivity and future climate scenarios.
