# Photo Geotagging

Geotags time-lapse bottom photos using a GPX track recorded simultaneously with a GNSS device, writing GPS coordinates into each image's EXIF metadata.

## Contents

| File | Description |
|------|-------------|
| `Geotag_Bottom_Pictures.ipynb` | Google Colab notebook — main processing workflow |
| `photo_gpx_geotagging.py` | Core Python script called by the notebook |

## How to use

Open `Geotag_Bottom_Pictures.ipynb` in Google Colab and follow the step-by-step instructions. No local Python installation required.

## Full documentation

See the [full documentation](https://seastrong-field-tools.readthedocs.io/en/latest/photo_geotagging.html) for field procedures, workflow description, and video tutorial.

## SEASTRONG context

This tool supports **Task 1.2** and **Task 1.3** of the SEASTRONG project (HORIZON-CL6-2025-01-BIODIV-07), contributing to habitat mapping and biodiversity characterisation within cross-shelf corridors.
