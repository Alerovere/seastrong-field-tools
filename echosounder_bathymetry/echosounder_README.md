# Echosounder Bathymetry

Processes depth data from a Deeper Pro echosounder and applies tide corrections using a reference tide gauge dataset.

## Contents

| File | Description |
|------|-------------|
| `Process_Echosounder_Data.ipynb` | Google Colab notebook — main processing workflow |
| `bathymetry_tide_processing.py` | Core Python script called by the notebook |

## How to use

Open `Process_Echosounder_Data.ipynb` in Google Colab and follow the step-by-step instructions. No local Python installation required.

## Full documentation

See the [full documentation](https://seastrong-field-tools.readthedocs.io/en/latest/echosounder_bathymetry.html) for field procedures, workflow description, and video tutorial.

## SEASTRONG context

This tool supports **Task 1.2** and **Task 4.5** of the SEASTRONG project (HORIZON-CL6-2025-01-BIODIV-07), contributing to bathymetry mapping of cross-shelf corridors and hydrodynamic modelling of coastal protection.
